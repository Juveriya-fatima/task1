# Netflix Data Cleaning and Analysis

## Project Objective

The objective of this project is to clean and preprocess the Netflix Movies and TV Shows dataset and perform basic exploratory data analysis using Python and Pandas.

## Dataset Used

Netflix Movies and TV Shows Dataset from Kaggle.

## Tools and Technologies Used

* Python
* Pandas
* Matplotlib
* Google Colab
* GitHub

## Data Cleaning Steps Performed

* Checked dataset structure and data types
* Identified and handled missing values
* Standardized column names
* Converted date columns into proper datetime format
* Checked duplicate rows
* Cleaned inconsistent formatting

## Exploratory Data Analysis

The following analysis and visualizations were performed:

* Movies vs TV Shows Distribution
* Top 10 Countries with Most Netflix Content
* Ratings Distribution
* Netflix Content Release Trend Over Years

## Key Insights

* Netflix contains more Movies than TV Shows.
* United States has the highest number of Netflix titles.
* TV-MA is one of the most common content ratings.
* Netflix content growth increased rapidly after 2015.

## Conclusion

The dataset was successfully cleaned and preprocessed using Python and Pandas. Exploratory Data Analysis helped identify important trends and patterns in Netflix content. The cleaned dataset is now ready for further analysis and visualization.
